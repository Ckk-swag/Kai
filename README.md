# 开开的科研成长计划 4.0

这是 GitHub Pages + Supabase 云同步版本。4.0 新增所有任务视图、任务完成鼓励动画与音效、周报月报里的 AI 生成周报/月报，并保留计划日期、任务筛选、项目进度、专注计时、精力量化折线图、分类想法、每日/手动马卡龙主题和登录后自动云同步等实用功能。

## 使用顺序
1. 直接把仓库根目录的 `index.html` 部署到 GitHub Pages 使用。
2. 在 Supabase 创建项目。
3. 在 SQL Editor 运行仓库根目录的 `schema.sql`。
4. 如需 AI 复盘，部署 `supabase/functions/ai-review` Edge Function。
5. 在网页“云同步”里填写 Project URL、anon public key、邮箱、密码。
6. 注册/登录后会自动读取云端最新内容；之后每次保存、完成、删除、顺延或归档都会自动推送云端。

## 新增实用功能
- 今日计划支持单独一行的大号任务动作输入框、计划日期、关键词搜索、类型筛选，并可查看今日待办、所有任务、全部未完成、今日已完成和全部已完成。
- “所有任务”会把未完成任务排在上方、已完成任务沉到下方；全部已完成会按类型归类展示，任务类型新增“运动”。
- 预制任务新增模型构建、体育锻炼、人际交往等快捷入口，任务可一键顺延到明天或下周。
- 首页显示今日完成率、本月完成率，以及逾期叠加、本周待办、连续打卡、今日专注等小指标。
- 项目进度页支持自主新增论文、专利、课题、比赛、课程等多个长期项目。
- 右侧专注计时支持 15/25/50 分钟专注轮，并按日期记录专注事项、分钟数与开始/结束时间。
- 今日打卡精力改为 0/25/50/75/100%，状态选项扩展为带图标的稳定、平静、高效、开心、焦虑、紧张、卡住、疲惫、低落和兴奋，保存后自动生成本月精力量化折线图。
- 每日想法支持按科研、健身、成长三类分别记录并进入复盘。
- 侧栏鸡汤文案每小时更新，页面支持每日自动或手动选择多套马卡龙色系，按钮、卡片、边框、进度条和提示状态会一起跟随主题色变化。
- 今日计划输入区优化为任务动作独占一行、下方控件铺满整行，手机端任务按钮收在内容下方更整洁。
- 复盘归档支持选择日期生成、保存和读取，便于后续按日期回看。
- 周报/月报生成后可一键复制，并汇总周期内专注投入。
- 任务完成时会弹出鼓励动画并播放轻量提示音，让完成动作更有反馈。
- AI 生成周报/月报已合并到周报/月报页：前端只请求自己的 Supabase Edge Function，DeepSeek / OpenAI 兼容接口由后端函数代调用，并提供后端测试通道。
- AI 供应商、Base URL、模型和 API Key 可保存到当前 Supabase 账号，换设备登录后可读取；AI 生成的周报/月报会按日期和周期自动归档，同一天同周期重复生成会覆盖更新，输出为普通中文文本而不是 Markdown。
- 手机端把今日计划、项目进度、复盘归档、周报AI和云同步入口固定在屏幕底栏，推送云端、拉取云端、导出全部也固定在底部快捷区，方便单手切换和操作。
- 登录 Supabase 后自动推送本地改动，换设备刷新网页会自动读取云端最新内容。

## 注意
- 不需要 Render。
- 不需要 server.js。
- 不要把 service_role key 填进前端；AI API Key 也不要硬编码进前端代码。
- anon public key 可以放前端，但必须配合 RLS；schema.sql 已经配置 RLS。
- AI 复盘请通过 `supabase/functions/ai-review` 调后端，不要让浏览器直接请求 DeepSeek。
