# 开开的科研成长计划 3.8

这是 GitHub Pages + Supabase 云同步版本。新增计划日期、任务筛选、全部已完成归类、运动类型、顺延、项目进度、专注计时、专注记录、连续打卡、本月完成率、精力量化折线图、分类想法、每日马卡龙主题、每小时鸡汤和登录后自动云同步等实用功能。

## 使用顺序
1. 把 public/index.html 上传到 GitHub Pages，或直接重命名为 index.html 使用。
2. 在 Supabase 创建项目。
3. 在 SQL Editor 运行 supabase/schema.sql。
4. 在网页“云同步”里填写 Project URL、anon public key、邮箱、密码。
5. 注册/登录后会自动读取云端最新内容；之后每次保存、完成、删除、顺延或归档都会自动推送云端。

## 新增实用功能
- 今日计划支持单独一行的大号任务动作输入框、计划日期、关键词搜索、类型筛选和查看全部未完成/今日已完成/全部已完成。
- 全部已完成会按类型归类展示，任务类型新增“运动”。
- 预制任务新增模型构建、体育锻炼、人际交往等快捷入口，任务可一键顺延到明天或下周。
- 首页显示今日完成率、本月完成率，以及逾期叠加、本周待办、连续打卡、今日专注等小指标。
- 项目进度页支持自主新增论文、专利、课题、比赛、课程等多个长期项目。
- 右侧专注计时支持 15/25/50 分钟专注轮，并按日期记录专注事项、分钟数与开始/结束时间。
- 今日打卡精力改为 0/25/50/75/100%，保存后自动生成本月精力量化折线图。
- 每日想法支持按科研、健身、成长三类分别记录并进入复盘。
- 侧栏鸡汤文案每小时更新，页面每天切换一套马卡龙色系。
- 复盘归档支持选择日期生成、保存和读取，便于后续按日期回看。
- 周报/月报生成后可一键复制，并汇总周期内专注投入。
- 登录 Supabase 后自动推送本地改动，换设备刷新网页会自动读取云端最新内容。

## 注意
- 不需要 Render。
- 不需要 server.js。
- 不要把 service_role key 填进前端。
- anon public key 可以放前端，但必须配合 RLS；schema.sql 已经配置 RLS。
